go-gameday
==========

Just getting going here.

Right now, will just go get the same game each time you run and print some stuff to the console.

Prerequisites: go

To build:
---------

    git clone git@github.com:ecopony/go-gameday.git
    cd go-gameday
    go build gameday.go
    ./gameday
